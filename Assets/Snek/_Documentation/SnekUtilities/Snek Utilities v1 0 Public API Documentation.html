Attributes
  • [UseSnekInspector]
    • Can be used on a child class of SnekMonoBehaviour or SnekScriptableObject.
    • Changes the looks of the component’s inspector to a custom style.
    • You can also disable the component’s Inspector value changes by using the [UseSnekInspector(true)] overload.
      The bool parameter inspectOnly disables the drawn GUI of the entire component’s inspector when set to true.

  • [SnekAutoGenerateAsset]
    • Can be used on a SnekScriptableObject child class to ensure the asset always gets generated if missing
    • Assets which use this attribute can be easily referenced for editor-only workflow by SnekScriptableObjectManager.GetAsset<T>() method

  • [SnekInspectOnly]
    • Can be used on [SerializeField] or public fields to expose them in the inspector while preventing their values from being changed through the inspector.
      This is recommended for simple types only (int, float, string, Vector3, etc.)
    • Note: You can use it on [Serializable] classes/structs as well, but it can provide unwanted behaviour depending on the value types inside.
      For that use case, you can add the attribute to each value inside a class/struct, or disable the entire GUI using the [UseSnekInspector(true)] overload on the SnekMonoBehaviour using it instead.
    • Note: Incompatible with List<T>, use [UseSnekInspector(true)] for the SnekMonoBehaviour using it or block the GUI through custom inspector logic instead
